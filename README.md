# Автоматический черный список (Blacklist) для VPN

Репозиторий содержит актуальные списки доменов и IP-адресов, доступ к которым заблокирован или ограничен. Эти ресурсы направляются **внутри VPN-туннеля** для восстановления доступа, либо используются в блокировщиках рекламы. Все данные обновляются автоматически **каждый час**.

---

## 🚀 Особенности проекта

* **Частота обновления:** Каждый час (24/7) без участия человека.
* **Автоматизация:** Сборка, фильтрация и проверка данных работают на базе GitHub Actions.
* **Оптимизация:** Списки очищены от мусора, дубликатов и мертвых доменов для снижения нагрузки на ваш клиент.

> [!NOTE]
> Списки обновляются на стороне GitHub автоматически. Если вы один раз подключили URL-ссылку в свое приложение, ваш клиент будет сам скачивать свежую версию по расписанию.

---

## 🛠 Инструкция по подключению

Для настройки вашего VPN-клиента или блокировщика рекламы используйте прямые ссылки (`Raw`).

### Как получить ссылку:
1. Откройте нужный файл списка в этом репозитории.
2. В верхнем правом углу текстового блока нажмите кнопку **Raw**.
3. Скопируйте получившийся URL-адрес из адресной строки браузера.

### Поддерживаемые клиенты:
* **Sing-box / Xray / v2ray:** Скопированная ссылка вставляется в блок удаленных правил (`remote rules`) для маршрутизации через прокси.
